# koacms 内容管理系统框架
koa + node + mongodb

内容管理系统：
连接本地mongodb数据库：mongodb://127.0.0.1:27017

开启本地数据库命令：
  mongod -f /Users/username/mongodata/etc/mongo.conf
	tail -f /Users/username/mongodata/logs/mongodb.log
  
node热部署工具：supervisor

模版引擎：art-template

获取表单数据： koa-bodyparser

session：koa-session

静态文件中间件： koa-static

加密： md5
